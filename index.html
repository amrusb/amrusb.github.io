who am i

Hello! I'm Bartosz, a -year-old computer science student at the Cardinal Stefan Wyszyński University in Warsaw, living life to the fullest. I have a profound passion for code, sounds, and creativity, which together make up my extraordinary journey through life.

Deeply immersed in the world of programming, I'm like an artist who creates virtual masterpieces in the language of Java. Every line of code is an opportunity for me to express my creativity and achieve something new. However, my passion extends beyond just coding...

The rhythm of music beats in my heart, which I create in the choir. Singing is a way for me to express emotions and connect with others in the harmony of sounds. But my artistic interests don't stop there. In my free time, I engage in computer graphics, creating posters and visual stories. Designing websites is another area where I devote myself to creativity and exploring new possibilities.

And when I need a moment of peace and contemplation, I head out to mountain trails, cycling routes, or swim. It's in nature that I find balance and inspiration, which I later translate into my projects.

designing

Expense menager

This program allows the user to control their budget by managing expenses. The application uses a relational database to collect, search and process information about the user's expenses. The program allows the user to make changes to the database by adding, editing or deleting a record corresponding to the user's expense or account.

Image Segmentation

This program showcases diverse methods for image segmentation, employing algorithms like k-means, region growing, Otsu thresholding, and adaptive thresholding with an integral image. It automates image partitioning into segments, enabling users to extract and analyze image components like objects, background, and contours.

Snake

Snake is a classic game where you command a snake that maneuvers across the board. Your objective is to devour fruits, enabling the snake to grow. However, be cautious not to collide with its own tail or strike the wall, as this signifies the game's end. With each consumed fruit, the snake elongates, introducing more challenges and complexities.

Mastermind

This is a strategic logic game that presents you with the challenge of uncovering a secret code, which is the sequence of colors hidden by opponent. Using the clues provided after each attempt, you must deduce and experiment to unveil the correct color combination. The game requires observation, logical thinking, and the skill of deduction.

Calculator

This app allows to perform basic mathematical operations, convert currencies using current exchange rates, and switch between temperature units using this convenient tool.
